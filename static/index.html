    GPIO MODE SWITCH SETUP

    Configure which BCM GPIO pins your SPDT mode switch uses for READ and WRITE. The pins are treated as active-low (internal pull-up; switch shorts to GND).

    Physical switch
    If OFF, the app forces WRITE mode (useful for dev systems without GPIO). If ON and pigpio/rppal access fails, the app will fall back to OFF mode.
    Read mode pin
    Switch throw that grounds this pin → READ mode.
    Write mode pin
    Switch throw that grounds this pin → WRITE mode.
    Wiring reminder: center/common → GND, one throw → READ pin, the other throw → WRITE pin. Center position (neither grounded) = OFF.
